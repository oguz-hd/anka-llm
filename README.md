# Gemini AI Asistanı - anka-LLM

Bu proje, React ve Node.js ile geliştirilmiş bir yapay zeka asistanıdır.

## Nasıl Çalıştırılır?

Projeyi çalıştırmak için aşağıdaki adımları izleyin:

### 1. Server'ı Başlatın
```bash
cd server
npm install
node server.js
```

### 2. Client'ı Başlatın
Yeni bir terminal açın ve aşağıdaki komutları çalıştırın:
```bash
cd client
npm install
npm run dev
```

Server `http://localhost:8020` adresinde, client ise `http://localhost:5174` (veya başka bir uygun port) adresinde çalışacaktır.
